Installation
------------

1. Place these files in modules/project:
     project.module
     project.install
     project.update
     project.css
     project.inc
     release.inc
     issue.inc
     comment.inc
     mail.inc

2. Enable the project module under administer >> modules

3. Grant the proper access to users under administer >> access control


Upgrading
---------

0. Make sure you backup your database!

1. Copy new files to modules/project.

2. Login as the site administrator (user #1) and run the update.php script.

3. Configure the status messages at admin/settings/project/status
   and admin/access.
